Description: How to Register Account in MSME Web Portal

Home - Google Chrome
1.

Click Registration at Menu

2. Click To apply registration
https://localhost:44305/Users/Register - Google Chrome
3. Click Full name text box
4. Type "Name" in the text box.
5. Click Email text box
6. Type email in the text box.
7. Click mobile number text box.
8. Type mobile number in the text box.
9. Click the text box
10. Type password in the text box
11. Click the text box
12.

Re-type the password in the text box

13. Click Register button .
https://localhost:44305/Users/Activate - Google Chrome
14.

Click text box and type the received code from SMS

15. Type code in the control
16. Click Activate button .
End of simulation.
17. End of simulation.
